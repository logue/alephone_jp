Software name	:Map Damager One
Author			:HogePiyo
License			:GPL

Thank you for downloading this program!

The purpose of this program is to split (unmerge) any maps.

With this, you can get every levels of a map.

--
How to use.

1.launch this program

2.select map file you want to unmerge

3.then, select directory for saving levels

4.completion dialog shown and click ok.

5.if warned error, click ok to skip it, or click cancel to quit program

ソフト名	:Map Damager One
作者		:HogePiyo
ライセンス	:GPL

ダウンロードありがとうございます。

このソフトはマップを各レベルごとに分解する目的で作っています。

MapEditorOneの保存機能を調整して形にしました。

標準マップならば遊べる程度に保存できるはずです。

他のサードパーティ系は未確認です。

ちなみに、MapEditorOneとMapViewerOne、そしてこのMapDamagerOneで
ファイル保存機構のバージョンが若干違います。

2007/02/08 現時点ではMapDamagerOneが最新かつ確実にMapデータを保存できるはずです。

他二つについてはそれぞれの次回リリースまでお待ちください。

--
使い方

1.起動するとファイル選択が出るので、マップファイルを選びます

2.次にディレクトリ選択が出るので、出力先を指定します。

3.完了ダイアログが出れば終了です。

4.エラーメッセージが出たら、okかcancelを押します。
 -ok	:スキップして続行します。
 -cancel:終了します。

--
未実装

-マージされているターミナルテキスト（スクリプト）の保存
-ターミナルピクチャの保存

ピクチャはリソースを除いたりする必要もあるので分かりませんが、
物理ファイルとターミナルテキストは取得できているのですぐできると思います。
