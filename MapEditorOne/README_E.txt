 Title              MapEditorOne
 Execution file     MapEditorOneSDI.exe
 spec               Windows that can execute AlephOne
 
--------------------------------------------------------------

-Ctrl+O (open)      load map
-"-"/"^","+"        zoom out/in
-Shift+MouseDrag    drag map
-Ctrl+MouseWheel    zoom in/out
-jump level         jump to another level
-terminal viewer    can see terminal datas
-Click              select an object/point/line.you can look object's informations in tool window
-Drag               drag an object/point/line
