The purpose of this program is to see any maps, levels, items, secrets, and so on.

Some functions seem to remain. Found them.
Some functions does not work, yet.

-Open Map Files(File menu or Open button)
-Move view(Ctrl+Drag or Hand-tool)
-Select and check item/polygon.(Click)
-Zoom in/out(Ctrl+Wheel or Magify-tool)
-Clear(File-New)
-Preferences(Edit-Preferences)
-Some dialogues(Show-***)
-Looks in Draw?/Polygon-Type/Height(Mode)
-Height Tool Dialog make elavation easy to see
-He never load shapes
-Jump level(Special-Jump)
-Level's info
-Object placements
-Terminal Viewer
