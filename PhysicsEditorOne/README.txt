●あいさつ
　この度はPhysicsEditorをダウンロードしていただきありがとうございました。
　BUNGIE社が昔MacでリリースしたFPS=Marathonの後継オープンソースソフトである
Aleph Oneは、マルチプラットフォームレイヤーであるSDLをベースにしているためWindows
やUNIXでも動作します。それだけでも十分楽しいのですが、アドオンである
ShapesやPhysicsFilesを編集することでより多彩なゲームを楽しむことができます。
　しかしMac版が元であるためWindowsで動作するツール群があまり作成されていません。
Mapエディタはひとつだけ存在しますが、Physicsエディタはどうやら誰も作っていない
ようです。

　ならば、と私HogePiyoはPhysicsEditorを自作しようと試みている訳です。

●インストール方法
　レジストリはいじっていません。解凍して好きな場所に設置することで完了します。
　実行にはMFCランタイムとSDL.DLLファイルが必要です。MFCランタイムについては
Visual Studio 2005用を
http://www.vector.co.jp/soft/win95/util/se390159.html
からダウンロードしてインストールしてください。
SDL.DLLは付属していますので何もしなくてよいです。

●アンインストール方法
　ディレクトリごと消してください。

●使用方法
　「PhysicsEditor.exe」をダブルクリックなどで起動すると「モンスター1(物理)」
タブが開いた状態となります。各項目をそれぞれ修正することでPhysicsFileを編集
することができます。項目にはコンボボックスから選ぶものもあります。
　また、タブをクリックすることで他の分野についても変更できます。
　Fileメニューにおいてファイルの入出力ができます。クローズボックスか、
Fileメニューの「終了」から本ソフトを終了します。
