#0 Greeting
    Thank you for downloading this software. With this, you can demerge maps 
which are merged by Forge, for example.
    Unmerged maps are free from any physics included in merged one. So you
can enjoy them with pure physics or your own physics.


#1 Installation
 Copy this directory to anywhere you like


#2 How to Use
 1.Select Map file you want to unmerge.
 2.Then, select directory that you want to save unmerged map elements.
 3.Completion dialog shown and click ok for exiting.

I'm in Pfhorums. If you have reports or requests, report there.

○挨拶
　ダウンロードしていただきありがとうございます。このソフトを使うと、
Forgeなどでマージされているマップデータを分解（Unmerge）し、一つ一つのマップに
分解できます。

　物理ファイルが無効になりますので、違う物理ファイルで遊びたいとき、また、
内容をコピーしたり改変したいときにでも使ってやってください。

PfhorumsとかMixiのコミュとかで何か言ってもらえればできる限りお応えします。
使用によるいかなる問題・損害に対して、HogePiyoは責任を負いません。

○使い方
１：ダイアログが出たら、マージされたマップファイルを選択します。
２：出力先のディレクトリを指定します。
３：完了ダイアログが出たら最後まで処理できています。
